CLI tool to generate complete, standards-compliant GitHub repository scaffolding for any project type, following GitHub Community Standards and best practices.

## Quick Start

### Prerequisites
- Python 3.12+
- Poetry

### Install
```bash
git clone https://github.com/valorisa/Github-Universal-Scaffolding-Generator-Advanced.git
cd Github-Universal-Scaffolding-Generator-Advanced
poetry install
```

### Usage
```bash
github-scaffolding-generator generate --project-name my-project --project-type cli --stack "Python 3.12 + Poetry"
```

## Installation (Multi-OS)
### Linux/macOS
```bash
pip install github-scaffolding-generator
```
### Windows
```powershell
py -m pip install github-scaffolding-generator
```

## Features
- Génération complète conforme aux GitHub Community Standards
- Support multi-types de projets (library, webapp, cli, etc.)
- Conventional Commits, SemVer, Keep a Changelog intégrés
- Workflows CI/CD adaptés à votre stack
